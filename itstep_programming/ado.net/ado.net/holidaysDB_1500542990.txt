Разработать приложение - пользовательский интерфейс для работы с базой данных.

Использовать SqlConnection для подключения к БД. Использовать WPF либо WinForms в качестве библиотеки графического интерфейса.

Использовать базу данных, разработанную для экзамена по MS SQL, либо разработать другую базу данных.

Пользовательский интерфейс должен обеспечивать
- Просмотр всех имеющихся в базе данных в удобной форме
- Добавление, удаление и изменение записей по меньшей мере в одну таблицу, для которой задано ограничение внешнего ключа
- Вызов хранимых процедур
